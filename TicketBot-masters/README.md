# TicketBot
A support discord.js bot


## Required

Discord.js
Some discord perm : 
  - Manage channel
  - Manage roles


## Usage

  - PM the bot and it's going to create a channel with the message
  - !close in the channel of the ticket


## Config

Put
  - Your server ID
  - Your token
  - The moderator roles
